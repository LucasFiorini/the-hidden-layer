# hello, world

Every blog needs a first post. This is mine.

## Why I built this

I've been meaning to have a corner of the internet to call my own for years. Not a social media profile, not a Substack — something I control, something that runs fast, something I built myself.

So here it is: a statically generated site built with Next.js, MDX for writing, and a dark cyberpunk aesthetic because why not.

## What I'll write about

- **Systems programming** — Rust, Go, C, all the low-level stuff I find fascinating
- **Distributed systems** — consensus algorithms, eventually consistent data stores, the fun problems
- **Tools I'm building** — notes and devlogs for projects in progress
- **Things I'm learning** — papers, books, ideas worth sharing

## The stack

```bash
next.js       # static site generation + routing
tailwind      # utility-first css
framer-motion # animations
mdx           # markdown + react components
```

Simple. No database, no server, just files and a CDN.

See you in the next one.
